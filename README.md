# Rayleigh - Infrastructure 개인 학습용 저장소

## Terraform

https://www.terraform.io/downloads.html

## script usage

```sh
# mac
./terraform_setup.sh --setup -p | pbcopy
```

```sh
# windows
./terraform_setup.sh --setup -p | clip
```

```sh
# ubuntu
sudo apt-get install xclip -y;

./terraform_setup.sh --setup -p | xclip
```
